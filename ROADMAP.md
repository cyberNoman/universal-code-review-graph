## 🎓 For New Contributors

### Easy Wins (Good First Issues):
1. **Add a new language** - Copy existing parser pattern
2. **Improve documentation** - Fix typos, add examples
3. **Write tests** - Increase coverage
4. **Translate README** - Help non-English speakers

### Medium Tasks:
1. **Add IDE plugin** - VS Code is done, add JetBrains
2. **Improve CLI** - Better output formatting
3. **Performance optimization** - Speed up graph building

### Hard Tasks:
1. **Cross-file import resolution** - Handle complex dependencies
2. **WASM compilation** - Browser-native version
3. **Incremental parsing** - Only re-parse changed files
